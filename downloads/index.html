Windows

Tested on Windows 7 and (briefly) Windows 8.1, but should be compatible all the way back to Windows XP.

The Standalone version is a folder full of DLLs and two EXEs you can put anywhere.
The Installer is what it says on the tin - a convenient installer that installs the program and makes Start Menu shortcuts.

Source

If you want to build the project yourself, you can either get the latest sources from the GitHub Repository, or download a snapshot of the source as of this release below.

Build Requirements:

  • Qt 5
  • A C++11 compatible compiler
    (up-to-date GCC, Clang or MinGW, or Microsoft Visual Studio 2013)
To build the Windows installer, you'll also need InnoSetup 5 (with the plugin the installer asks you about) installed.
